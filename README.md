A clean and responsive **CRUD-based To-Do Manager** that allows users to add, edit, delete, and store tasks in an organized and user-friendly interface.

---

## ✨ Features

- 📱 Fully responsive design for all devices.
- 🎨 Clean and modern UI.
- ➕ Add new tasks easily.
- ✏️ Edit existing tasks.
- 🗑️ Delete tasks when completed or no longer needed.
- 💾 Store tasks for later use.
- ⚡ Smooth and simple task management.
- 🛠️ Easy to customize and maintain.

---

## 🛠️ Tech Stack

- **React** — UI development.
- **Tailwind CSS** — Styling and layout.
- **JavaScript** — Interactive functionality.
- **HTML5** — Semantic structure.
